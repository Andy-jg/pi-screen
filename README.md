pi-screen
=========

本程序用于树莓派通过GPIO连接1602的效果演示，使用wiringPi库和其附带的LCD库进行GPIO和1602的操作。

qtsharp

实例1： 
$sudo ./l602    
屏幕显示：      
    ------------------    
    | Raspberry Pi!  |    
    |51.9C 215/477MB |    
    ------------------    

实例2： 
$sudo ./1602 QtSharp    
屏幕显示：  
    ------------------      
    |QtSharp         |      
    |51.9C 215/477MB |      
    ------------------      

实例3： 
$sudo ./1602 \ \ Hello\ World   
屏幕显示：  
    ------------------      
    | hello world    |  
    |51.9C 215/477MB |      
    ------------------
